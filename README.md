# cyclistic-bike-share-analysis
Google Data Analytics Capstone – Cyclistic Case Stud

## Business Problem
The objective of this analysis is to understand how casual riders and annual members use Cyclistic bikes differently and to identify strategies to convert casual riders into annual members.

## Data Source
Historical bike trip data provided by Cyclistic, including ride start time, end time, rider type, and trip duration.

## Data Cleaning & Preparation
- Removed rides with zero or negative duration  
- Handled null and blank values  
- Created a new feature `ride_length_minutes`  
- Extracted day of week and hour from timestamps  

Data cleaning was performed using Excel.

## Key Insights
- Casual riders take longer rides on average  
- Casual riders primarily use bikes on weekends  
- Annual members ride consistently during weekdays, indicating commuting behavior  
- Member usage peaks during morning and evening hours  

## Visualization
Dashboards were created using Tableau to visualize:
- Average ride length by rider type  
- Day-of-week usage patterns  
- Time-of-day trends  

Dashboard images are included in this repository.

## Recommendations
- Promote weekend-focused membership plans  
- Offer trial or short-term memberships to casual riders  
- Target marketing campaigns during high weekend usage periods  

## Tools Used
- Excel – data cleaning and feature engineering  
- Tableau – data visualization and dashboard creation
